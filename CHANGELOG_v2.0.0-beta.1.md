# SproutG Desktop v2.0.0-beta.1

## Изменено

- Верхняя панель переведена на русский язык.
- `Bridge/Ready` заменён на короткий русский статус подключения к Google Таблице.
- Кнопка перезагрузки UI/моста перенесена из верхней панели в окно настроек.
- `Компании` перенесены в верхнюю панель.
- Из внутренних переключателей страниц убраны `Компании` и `Настройки SproutG`.
- Добавлен расширяемый список тем:
  - Dark Classic;
  - Light Classic;
  - Dark iOS;
  - Light OldMoney.
- Исправлена светлая верхняя панель в Light Classic.
- Добавлен общий индикатор синхронизации/загрузки для bridge-запросов.
- Автоустановка обновлений не запускается в фоне. Проверка при старте выполняется не чаще одного раза за загрузку Windows и показывает только уведомление.
- Очистка кеша больше не удаляет site storage/cookies, чтобы не сбивать Google-сессию.

## Чистка

- Удалены старые файлы Electron-wrapper renderer:
  - `src/viewPreload.js`;
  - `src/renderer/index.html`;
  - `src/renderer/renderer.js`;
  - `src/renderer/styles.css`.
- Удалены старые BrowserView/postMessage helpers из main process.
